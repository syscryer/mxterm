# 终端完整分屏

## Goal

让 mXterm 的终端子 Tab 支持完整分屏能力。分屏组本身是一个普通子 Tab，可以和未加入分屏的普通终端 Tab 正常切换；一个 pane 绑定一个真实终端会话，支持跨 SSH、本地、Telnet、Serial 会话组合。

## User Value

- 用户可以同时观察多个终端，又不会失去原有子 Tab 切换习惯。
- 分屏成员、普通终端 Tab 和 `+` 新建入口的归属清晰，不出现“看得见但点不动”的标签。
- 进入、退出和切换分屏不会销毁 xterm、丢失滚屏或漏掉实时输出。

## Requirements

- 普通终端 Tab 保持现有命名和编号规则；点击 `+` 继续按现有序号创建新终端，已有标题不重排、不复用。
- 从普通终端创建分屏时，当前 Tab 原位置转换为分屏组 Tab。
- 分屏组成员不再重复显示为普通子 Tab；没有加入分屏的普通 Tab 必须继续保留并可正常切换。
- 点击分屏组外的普通 Tab 时离开分屏视图；点击分屏组 Tab 时恢复布局和上次聚焦 pane。
- 支持向右分屏、向下分屏和四分屏，最多四个 pane。
- 分屏布局菜单提供默认勾选的“同会话”选项；勾选时自动按当前 SSH 连接、本地 profile、Telnet 或串口连接新建终端并填入新 pane。
- 取消勾选“同会话”后，新 pane 保持为空并立即打开顶部会话选择器；方向分屏取消选择时撤销该空 pane。
- 四分屏保留已有 pane 绑定，空位按左上、右上、左下、右下顺序提供选择入口。
- pane 可以绑定已有 SSH、本地、Telnet、Serial 会话，也可以新建同连接 SSH 或默认本地终端。
- 空 pane 的候选列表包含尚未打开会话的已保存 SSH 连接；选择后连接准备、认证、主机密钥确认、错误与重试流程都在当前 pane 内完成，不切换为全屏连接页。
- 同一个运行会话同一时间只能属于一个普通 Tab 或一个分屏 pane，不能重复挂载 xterm。
- 已经绑定到其他 pane 的运行会话不再出现在当前 pane 的会话候选列表中。
- pane 会话选择器提供搜索，可按连接名称、用户名、主机地址、端口和终端标题过滤已有会话与未打开 SSH 连接。
- pane 标题栏显示会话图标、名称、状态、搜索、清屏和关闭操作。
- 分屏组工具区提供布局、同步输入和均分操作；不提供解散分屏、批量转独立 Tab 或移出 pane。
- 同步输入以当前聚焦 pane 为主输入，可选择参与同步的 pane；界面快捷键不广播。
- 关闭 pane 时关闭对应终端会话并收缩布局；只剩一个 pane 时自动退化为普通终端 Tab。
- 关闭分屏组 Tab 时关闭组内会话，多条活动连接必须确认；组外普通 Tab 不受影响。
- 分隔线支持鼠标拖拽、方向键微调和双击均分，并设置最终 pane 的最小像素宽高。
- pane 编号、DOM 顺序和键盘顺序按左上、右上、左下、右下排列。
- 终端实例生命周期绑定会话，不绑定普通/分屏渲染分支；切换视图不能重新挂载 `TerminalPanel`。
- 所有视觉使用现有 Lucide、Radix、共享组件和 `--mx-*` token，同时覆盖亮色、暗色和 system-dark。
- 不引入启动首屏重依赖，不把 xterm 静态提升到更早加载路径。

## Acceptance Criteria

- [ ] 分屏组作为子 Tab 可与组外普通终端 Tab 双向切换。
- [ ] 分屏成员不会重复出现在普通子 Tab 列表，组外 Tab 和 `+` 编号规则保持不变。
- [ ] 向右、向下和四分屏保留已有绑定；默认自动创建同会话终端，取消勾选后自动打开空 pane 选择器。
- [ ] SSH、本地、Telnet、Serial 会话可跨来源组合，同一会话不会重复挂载。
- [ ] 未打开会话的 SSH 连接可从空 pane 直接发起连接，加载和错误状态保持在该 pane 内。
- [ ] 切换普通/分屏 Tab 不丢失滚屏、选择、搜索状态或实时输出。
- [ ] pane 搜索、清屏和关闭只作用于对应会话。
- [ ] 同步输入可见、可关闭、可选择目标，并在离开分屏组时自动关闭。
- [ ] 关闭 pane 后布局收缩，单 pane 自动退化为普通 Tab。
- [ ] 分隔线在桌面常用尺寸下不能把最终 pane 压缩到不可用宽高。
- [ ] 亮色、暗色和 system-dark 下标题栏、焦点、同步状态、空 pane 和分隔线清晰一致。
- [ ] `npm run check` 与终端相关源码检查脚本通过。

## Out of Scope

- RDP / VNC 分屏。
- 远程文件编辑器作为终端 pane。
- 同一 PTY 会话镜像到多个 xterm。
- 解散分屏、批量转换为独立 Tab、将 pane 移出为 Tab。
- 持久化复杂分屏布局到配置文件。
- 跨窗口拖拽 pane。
